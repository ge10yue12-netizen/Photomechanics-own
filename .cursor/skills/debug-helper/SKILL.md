# Debug Helper — QtProject_1

## 触发

MSVC 编译失败、链接错误、LNK 未解析符号、Pylon 运行时异常、Qt 插件/平台错误、相机打不开、程序崩溃。

## 流程

1. 阅读完整报错（输出窗口、`Log/run_*.log`、控制台）
2. 提取关键一行（错误码、符号名、文件行号）
3. 分类：

| 类型 | 本项目常见原因 |
|------|----------------|
| Pylon | SDK 未装、环境变量、`PylonInitialize` 失败、设备未找到 |
| Qt | MOC/UIC 未生成、Qt 路径错误、平台插件缺失 |
| MSVC | `/std`、运行时库不匹配、字符集、包含目录 |
| 链接 | Pylon `.lib` 未链、Qt 模块缺失、Debug/Release 混用 |
| 路径 | `Images/`、工作目录、相对路径与 `SavePathHelper` |
| 运行时 | 空帧、队列满、线程竞态、`copyLatestImage` 失败 |

4. 禁止首选「重装全部环境」
5. 最小修复：单变量变更，每步可验证
6. 命令注明在「x64 Native Tools」或 VS 开发者命令提示符下执行

## 输出格式

### 错误核心

最关键报错原文或摘要。

### 可能原因

按概率排序。

### 解决步骤

分步修复；含需检查的 VS 项目属性项（包含目录、库目录、附加依赖项）。

### 需用户补充的信息

缺日志、Pylon 版本、相机型号等时列出。
