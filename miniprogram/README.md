# PhotoMech 遥控小程序

手册：[../docs/BLE_REMOTE_GUIDE.md](../docs/BLE_REMOTE_GUIDE.md)

## WiFi（推荐）

1. PC 日志复制 `HTTP 遥控已启动：http://192.168.x.x:18765` 中的 **IP:端口**
2. 勾选 **不校验合法域名**，点 **预览** 扫码
3. 填地址与口令（默认 `1234`），点 **连接**
4. 地址会本地记住，下次自动填充

**勿填** `172.27.x.x`（虚拟网卡）；IP 前三段须与手机 WiFi 一致（常见 `192.168.x.x`）。

## 交互说明

- 按钮为按压式交互：按下变暗下沉，执行中保持按下态
- 任意操作进行中全局互斥（连接、遥控、扫描不可并发）
- 按钮可用性与 PC 端一致（如采集中不可关相机、阶段中不可关相机）
- 连接后每 2 秒自动同步状态；断连立即显示「已断开」
- 错误信息固定区域展示，避免页面跳动

## BLE（可选）

真机预览 → 刷新列表 → 点选电脑（需 PC 蓝牙外设模式）
